# 💰 Your Money

**Your Money** é um gerenciador de finanças pessoais desenvolvido para oferecer controle total sobre receitas e despesas com uma interface limpa, moderna e extremamente intuitiva.

---

### 📋 Pré-requisitos

- **PHP 8.2+**
- **Composer**
- **Node.js & NPM**

### 🛠️ Passo a passo

1. **Clone o repositório:**

    ```bash
    git clone https://github.com/gustavocoimbradev/your-money.git
    cd your-money
    ```

2. **Como rodar localmente:**

    ```bash
    # Iniciar o ecossistema completo
    composer run dev
    ```

3. **Acesse o sistema:**
   Após a execução, o gerenciador estará disponível em:
    ```bash
    http://localhost:8000
    ```

---

## ✨ Funcionalidades

- **Controle Financeiro:** Cadastro e monitoramento de entradas e saídas.
- **Interface Responsiva:** Design otimizado para qualquer dispositivo utilizando Tailwind CSS.

---

## 🛠️ Tecnologias utilizadas

- **Backend:** [Laravel 12](https://laravel.com)
- **Frontend:** [Vue.js 3](https://vuejs.org) / [Inertia.js](https://inertiajs.com)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com) (Foco em UI/UX limpa)
- **Banco de Dados:** MySQL / SQLite
