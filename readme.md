# 💰 Your Money

**Your Money** é um gerenciador de finanças pessoais desenvolvido para oferecer controle total sobre receitas e despesas com uma interface limpa, moderna e extremamente intuitiva.

---

### 📋 Pré-requisitos

- **Docker**

### 🛠️ Passo a passo

1. **Clone o repositório:**

    ```bash
    git clone https://github.com/gustavocoimbradev/amora.git
    cd amora
    ```

2. **Como rodar:**

    ```bash
    docker compose up -d
    ```

3. **Acesse o sistema:**
    ```bash
    http://localhost:8000
    ```

---

## 🛠️ Tecnologias utilizadas

- **Backend:** [Laravel 12](https://laravel.com)
- **Frontend:** [Vue.js 3](https://vuejs.org) / [Inertia.js](https://inertiajs.com)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com)
- **Infraestrutura:** [Docker](https://www.docker.com) & [Docker Compose](https://docs.docker.com/compose/)
